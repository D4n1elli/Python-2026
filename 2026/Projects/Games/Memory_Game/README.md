# Memory game 📝🧠

## Description
The game is easy to understand: the program generates a random number inside the difficulty the user selected, and the goal is to memorize this number and type it again without looking back. Then, it outputs if you guessed or not, and if not, gives you the correct answer.

## How it works
First, a menu with 4 different difficulties appears: easy, medium, hard and custom. The custom one allows the user to customize the start and end of the interval. Then, it shows a random number inside it, and the user must memorize it to be able to win. When they are ready, digit 1 to proceed and the number is scrolled out of view. The goal now is to type the number they memorized.
After attempting, they can choose to play again by going back to the menu, or quit the game.
