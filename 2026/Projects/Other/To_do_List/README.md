# To-do list

## Description
A Python project made to store and manage tasks the user writtes in the terminal

## How it works
First the user is introduced to a menu with 4 options, add a task, remove a task, view all the tasks they have written or quit the program.
To add a task, digit the "add task" number and type the task they want.
To remove a task, a list with all the tasks will be shown and the user needs to digit the number of the task they want removed.
To view the task, the user digits the "view task" number and a list with the tasks will be shown.
